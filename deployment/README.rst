Deployment Guide
----------------

This guide contains information on how to deploy and setup a Digital Rebar environment.  Once setup, next steps can be found
in the :ref:`user_guide`.

The deployment guide provides three parts:

#. Deciding what and where to deploy
#. Tweaking configuration for initial bring up and beyond
#. Tools to do deployment

Walk through each section to get deploy a Digital Rebar admin node or as always :ref:`quick_start`.
